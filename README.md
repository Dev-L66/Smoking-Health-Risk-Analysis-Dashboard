
An interactive Power BI dashboard that analyzes smoking habits and associated health risk factors, providing insights into patient demographics, smoking behavior, and cardiovascular health indicators.

---

## 📌 Project Overview

This dashboard was built using Power BI to explore relationships between smoking status and health metrics such as cholesterol, hypertension, BMI, and age.

The project demonstrates data cleaning, transformation, DAX calculations, and interactive dashboard design.

## 🎯 Objectives

- Analyze smoking status distribution
- Compare smoking behavior by gender
- Monitor cholesterol and hypertension risk
- Study smoking duration and daily cigarette intake
- Build an interactive healthcare analytics dashboard

---

## 📈 Dashboard Features

- KPI Cards
  - Total Patients
  - Average Age
  - Average BMI

- Smoking Status Distribution

- Smoking Status by Gender

- Cholesterol & Hypertension Risk Analysis

- Smoking Duration vs Daily Cigarette Intake

- Interactive Navigation

- Slicers and Filters

---

## 📌 Key Findings

### Healthy Group
- 18–28-year-olds had the highest average smoking duration (10.87 years).
- Never smokers represented 40.9% of the population.
- Female patients outnumbered male patients (105 vs. 88).
- The largest variation between smoking duration and cigarettes per day occurred in the 29–38 age group.

### Damaged Group
- 18–28-year-olds showed the highest average smoking duration (13.0 years).
- Never smokers accounted for 51.3% of patients.
- Female patients outnumbered male patients (25 vs. 12).
- The greatest difference between smoking duration and daily cigarette consumption occurred in the 29–38 age group.
